# superstore-sales-visualization

# Task 2 - Data Visualization & Storytelling

## Objective
Create visualizations that convey a compelling story using Superstore sales data.

## Tool Used
Tableau Public / Power BI

## Dataset
Superstore.csv - contains sales, profit, region, category data

## Charts Created
1. Sales by Category (Bar Chart)
2. Monthly Sales Trend (Line Chart)
3. Sales by State (Map)
4. Profit by Segment (Donut Chart)
5. Sales vs Profit (Scatter Plot)
6. KPI Summary Cards

## Key Insights
- Technology is the highest revenue category
- West region leads in sales
- Q4 shows peak sales performance
- Some sub-categories have negative profit margins
